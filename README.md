## Code for Age Gender recognition using Deep Learning 
### Gender, Age, Face detection using OpenCV and Caffe Model
#### Python
```
python AgeGender.py --input <input_file>(Leave blank for webcam)
```

### Usage with main.py
```
python main.py
```
Press `q` to exit the webcam window.
